How to Configure Post Monitoring
Option 1: Monitor Specific Posts

Copy post IDs from above and add them to config.py:

SPECIFIC_POST_IDS = [
    "{{ posts[0].id if posts else 'ABC123DEF' }}",
    "{{ posts[1].id if posts|length > 1 else 'XYZ789GHI' }}",
]
Option 2: Monitor All Posts

Enable monitoring for all posts:

MONITOR_ALL_POSTS = True
Option 3: Hashtag Filtering

Monitor posts with specific hashtags:

REQUIRED_HASHTAGS = [
    "#dmbot",
    "#automate",
]
Option 4: Caption Word Filtering

Monitor posts mentioning specific phrases:

REQUIRED_CAPTION_WORDS = [
    "dm for details",
    "link in bio",
]
Remember: After editing config.py, restart the web application for changes to take effect.
